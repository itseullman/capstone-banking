Odd Even Merge Sort

The odd-even merge sort algorithm is one of the two well-known parallel sorting algorithms developed by Dr. Batcher. It is based on a merge algorithm that merges two sorted halves of a sequence to a completely sorted sequence. However, the odd-even merge sort is not data dependent. This makes it possible to implement the algorithm as a sorting network. While the algorithm is not asymptotically optimal, Donald Knuth has concluded that compared to the AKS network, “Batcher’s method is much better, unless n exceeds the total memory capacity of all computers on earth!”
