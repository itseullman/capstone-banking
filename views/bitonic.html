Bitonic Merge Sort

The bitonic merge sort is a parallel algorithm for sorting devised by Dr. Batcher. It can be used as a construction method to build a sorting network. The resulting sorting networks consist of O(n log2(n)) comparators and have a delay of O(log2(n)).

Pseudocode:

1 for ( k = 2; k <= n; k * =2 ) // k is doubled every iteration

2    for ( j = k / 2; j > 0; j /= 2 ) // j is halved every iteration

3        for ( i = 0; i < n; i++ )

4            l = i ^ j

3            if ( l > i )

4                if ( ( i ^ k == 0 AND arr[ i ] > arr[ l ]) OR ( i ^ k != 0 AND arr[ i ] < arr[ l ]) )

5                    swap the elements in arr[ i ] and arr[ l ]
